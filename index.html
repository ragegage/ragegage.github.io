Stopify

MP3 Player built in Electron and Rails

Song metadata are read from .mp3 files and loaded into a local database. Title and artist information are stored, as is the path to the file. Songs are played, looked up, and put into playlists (including a built-in 'currently playing' queue) using the desktop app, built with Electron. File and folder structure is navigated to allow easy importing of big, nested file trees full of songs.

Chat Umbrella

A simple, authenticated, instant messaging app

This messaging app is an umbrella app composed of two separate projects: a chat server, and a web app with basic authentication. The chat server allows new "chatrooms" to be created, storing the rooms and their messages in memory. The web app uses Phoenix (especially its Channels and Presence features), Ecto (to persist user information), and Guardian (to manage user authentication).
